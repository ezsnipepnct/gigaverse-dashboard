# Gigaverse Dashboard

Real-time automated game bot with web interface for Gigaverse.

## Quick Start

### Backend (Python Bot)
```bash
cd gigaverse-backend
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

### Frontend (Next.js Dashboard)
```bash
npm install
npm run dev
```

### Access
- **Backend API**: http://localhost:8000
- **Web Dashboard**: http://localhost:3000
